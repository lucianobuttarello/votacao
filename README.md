# Funcionalidade

## Build
Para executar este projeto, execute na linha de comando:

``` mvn clean compile exec:java```

## O Swagger para acessar é o seguinte:

```http://localhost:8080/swagger-ui/index.html?configUrl=/v3/api-docs/swagger-config#/```
